# 機能要件

## 1. テーマ管理機能

| ID | 機能名 | 説明 |
|----|--------|------|
| F-001 | テーマ作成 | 新規テーマを作成する |
| F-002 | テーマ保存 | 作成・編集したテーマをローカルストレージまたはファイルに保存する |
| F-003 | テーマ読み込み | 保存済みテーマを読み込んで編集画面に表示する |
| F-004 | テーマ削除 | 不要なテーマを削除する |
| F-005 | テーマ一覧表示 | 保存済みテーマをリスト形式で表示する |
| F-006 | テーマ検索 | テーマ名やタグでテーマを検索・フィルタリングする |

---

## 2. テーマ編集機能

| ID | 機能名 | 説明 |
|----|--------|------|
| F-007 | グラフィカル編集 | カラーピッカーやスライダーを使用した視覚的な編集UI |
| F-008 | 色変更 | エディタの各要素（背景、文字、シンタックスハイライト等）の色を変更 |
| F-009 | テーマ名変更 | テーマの名前を編集する |
| F-010 | リアルタイムプレビュー | 編集内容をリアルタイムでプレビュー表示する |

---

## 3. インポート/エクスポート機能

| ID | 機能名 | 説明 |
|----|--------|------|
| F-011 | テーマインポート | 既存のVSCode/Windsurfテーマファイル(.json)を読み込む |
| F-012 | テーマエクスポート | 作成したテーマをVSCode/Windsurf互換形式(.json)で出力する |

---

## 4. 画面一覧

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| SCR-001 | テーマ一覧画面 | 保存済みテーマのリスト表示、検索、新規作成 |
| SCR-002 | テーマ編集画面 | テーマの視覚的編集とリアルタイムプレビュー |

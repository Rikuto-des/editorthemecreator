# 非機能要件

## 1. パフォーマンス

| ID | 要件 | 基準 |
|----|------|------|
| NF-001 | 初期ロード時間 | 3秒以内 |
| NF-002 | 色変更反映 | 100ms以内にプレビューに反映 |
| NF-003 | テーマ保存 | 1秒以内に完了 |

---

## 2. ユーザビリティ

| ID | 要件 | 基準 |
|----|------|------|
| NF-004 | レスポンシブデザイン | デスクトップ・タブレット対応（最小幅: 768px） |
| NF-005 | アクセシビリティ | WCAG 2.1 AA準拠 |
| NF-006 | 直感的UI | 初回利用者が5分以内に基本操作を習得可能 |

---

## 3. 互換性

| ID | 要件 | 基準 |
|----|------|------|
| NF-007 | ブラウザ対応 | Chrome, Firefox, Safari, Edge最新2バージョン |
| NF-008 | テーマ形式 | VSCode Theme JSON形式に完全準拠 |
